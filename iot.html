物联网交互 · 智能安全距离预警装置

INTRO

Project Overview

项目概述

This project is a home-based intelligent safety protection IoT device designed for elderly people with mild cognitive impairment. Different from traditional single sensor devices, it constructs a complete edge-side IoT interaction closed loop: environmental perception → data operation → intelligent judgment → human-computer interaction → state self-reset.

With Arduino UNO as the main control core, the device collects environmental data through ultrasonic sensors and completes risk judgment via local intelligent algorithms. It finally realizes low-stimulation human-computer interaction through screen visualization and dynamic light prompts, operating fully automatically without manual supervision.

01

Overall IoT Interaction Architecture

整体物联网交互架构

As a lightweight edge-side intelligent IoT interaction system, the device consists of four layers: the perception collection layer, the core operation layer, the interaction execution layer, and the state reset layer.

The coordinated operation of all layers realizes fully automatic interaction from environmental data collection to intelligent feedback without manual intervention, which is suitable for long-term standby monitoring in home scenarios.

02

Step-by-Step Complete Interaction Operation Process

分步完整交互操作流程

1. Perception Collection Layer: Real-Time Environmental Data Collection (Input Interaction)

After power-on and standby, the HC-SR04 ultrasonic sensor works continuously. It emits ultrasonic signals at a fixed frequency and receives echoes. By calculating the round-trip time of sound waves, it collects real-time distance data between people and dangerous areas such as window sills, stoves and balconies, converts analog physical signals into identifiable digital signals, and transmits the data to the Arduino UNO main control chip in real time.

As the entry of IoT interaction, this process realizes the conversion from physical environment to digital data, endowing the device with the ability to perceive environmental changes and providing data support for subsequent intelligent judgment.

2. Core Operation Layer: Intelligent Data Judgment and State Identification (Processing Interaction)

The main control chip conducts intelligent logic judgment through built-in program algorithms after receiving real-time distance data, which is the core of the device's intelligence. The system presets two judgment thresholds: a dangerous distance of 60 cm and a dangerous residence time of 3 seconds.

When the detected distance between the human body and the dangerous area is ≤ 60 cm, the system starts timing and continuously monitors the residence state. If the person does not leave within 3 seconds, the system judges it as a dangerous residence state and triggers an early warning instruction. If the person leaves quickly and the distance returns to the safe range, the system clears the timing data automatically and maintains the safe standby state.

Different from the single data feedback of traditional sensors, this process realizes active thinking and logical decision-making of IoT devices, avoiding false triggers caused by transient approaching and improving the accuracy of the device.

Dangerous Distance: ≤ 60 cm
Dangerous Residence Time: ≥ 3 seconds

3. Interaction Execution Layer: Low-Stimulation Intelligent Human-Computer Feedback (Output Interaction)

Once the system identifies a dangerous state, it activates coordinated human-computer interaction of two modules. Adapting to the sensory characteristics of people with cognitive impairment, it adopts gentle intelligent feedback to avoid anxiety and panic caused by harsh alarms of traditional equipment, forming a humanized IoT interaction experience.

Screen Visual Interaction

The 0.96-inch OLED display immediately refreshes the interface and exits the standby state. It shows the prompt "Please stay away from dangerous areas" in large and concise centered fonts. The intuitive text conforms to the visual habits of the elderly and realizes effective visual safety reminders.

Dynamic Light Interaction

The WS2812B light ring starts a warm-yellow breathing gradient effect with gradually changing brightness. The soft and non-glare light provides continuous ambient reminders without noise, achieving low-stimulation long-term safety prompting.

4. State Reset Layer: Adaptive Closed-Loop Regulation (Cyclic Interaction)

The device supports intelligent self-reset and closed-loop regulation. It keeps monitoring the environmental state after triggering early warnings. When the user leaves the dangerous area and the detection distance returns to over 60 cm (safe range), the system clears the timing data, turns off the early warning state, extinguishes the light ring, and restores the screen to the standby interface showing "Safety Monitoring". The device then returns to the normal monitoring mode.

This layer realizes dynamic state switching and fully automatic cyclic operation. No manual restart or reset is required, enabling 24-hour unattended intelligent safety protection.
